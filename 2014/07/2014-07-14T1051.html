Disabling Fiddler Proxy on Win 8

If you're like me, and I know I am, you installed Fiddler by Telerik to test your Web API responses. Eventually you get around to closing out fiddler and find that you can no longer browse the web. You stumble around some trying out various solutions, but eventually you turn fiddler back on or you are using another computer to read this blog post.

Well you came to the right place. You need to disable the proxy that fiddler installed.

  1. Go to start. Type in network proxy settings. Click the that.
  2. Set Use a proxy server to Off

Remember to turn it back On when you use Fiddler again later.
